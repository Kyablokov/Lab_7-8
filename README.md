# Lab_7-8
Лабораторная работа 7-8
ФИО: Яблоков Николай Дмитриевич
Группа: СКБ-251

***Задание***
Написать консольную программу для создания односвязного списка и хранения в каждом его узле PCB (Process Control Block) с возможностью удаления и добавления процессов.
Реализовать класс процесса, узла односвязного списка, односвязного списка.
Реализовать контроль входных данных.


***Описание особенностей кода***
Струкутра `PCB` с полями и методами из ТЗ + методом для взятия статуса
Класс `ListNode` с полями и методами из ТЗ + нужные get и set методы
Класс `ProcessList` с полями и методами из ТЗ + явный деструктор
В файле "main.cpp" функции для удобного ввода чисел и строк
